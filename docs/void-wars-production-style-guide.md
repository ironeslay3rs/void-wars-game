# Void Wars — Production Style Guide

Purpose: convert the approved Void Wars visual direction into a build guide artists, UI designers, and future implementers can ship against without drifting the look.

Working terminology: **Pure/Rune** maps to the current in-project **Spirit / Rune / Ember** path language. Use the Pure/Rune label in concept communication when discussing the faction fantasy; use Spirit/Rune naming when matching current game content.

## 1. Faction Visual Law Sheet

### Bio
- **Silhouette rules**
  - Read from distance as predator-first, not soldier-first.
  - Keep asymmetry alive: one limb, shoulder, spine ridge, or jaw line should feel evolved beyond baseline human proportion.
  - Use forward-leaning posture, hooked profiles, tendon arcs, and stalking neck/head placement.
  - Mass should feel grown, not manufactured. Avoid clean bilateral armor reads.
- **Materials**
  - Wet chitin, stretched hide, scar tissue, sinew wraps, exposed growth plates, ossified spikes, containment straps.
  - Surfaces should feel adaptive and reactive, as if still changing under stress.
  - Hard materials may exist only as grafts, restraints, or scavenged field hardware.
- **Accent colors**
  - Core accent: toxic emerald.
  - Support accents: deep moss, surgical teal, bruised black-red in damaged areas.
  - Accent energy should sit inside the form like active tissue, not as painted trim.
- **Iconography**
  - Coil, fang, claw, specimen marks, growth rings, split-cell motifs, containment sigils.
  - Shapes should imply hunting, harvesting, and mutation.
- **FX behavior**
  - Pulsing internal glow, spore drift, gland venting, tissue twitch, vein-light surges.
  - FX should spread, creep, or breathe outward rather than burst mechanically.
- **Forbidden tropes**
  - No generic zombies.
  - No clean superhero biotech suits.
  - No cartoon slime blobs.
  - No neon graffiti monster styling.
  - No insectoid overload that erases the human-to-predator transition.

### Mecha
- **Silhouette rules**
  - Read as disciplined frame control: upright, deliberate, modular, load-bearing.
  - Favor strong chest blocks, shoulder plating, clear limb segmentation, and stable stance geometry.
  - Shapes should communicate engineered intent and tactical endurance before speed.
  - Complexity should stack in layers, not noise; every module must feel mounted for a reason.
- **Materials**
  - Brushed alloy, plated steel, machine ceramic, exposed fasteners, cable bundles, heat-scored casings, weapon housings.
  - Wear should show service life: abrasion, machining marks, carbon scoring, replaced modules.
  - Surfaces should feel manufactured and serviceable, never ornamental luxury chrome.
- **Accent colors**
  - Core accent: cold cyan.
  - Support accents: steel slate, blue-white diagnostics, restrained ember-orange for heat or forge events only.
  - Accent application should appear as system light, targeting logic, or power routing.
- **Iconography**
  - Core, frame, module grids, weapon rails, serial marks, discipline seals, precision reticles.
  - Symbols should feel institutional and engineered.
- **FX behavior**
  - Scanning sweeps, servo pulses, vent heat, calibration lines, controlled sparks, thin data-light passes.
  - FX should resolve in measured intervals, not chaotic magic storms.
- **Forbidden tropes**
  - No glossy consumer sci-fi.
  - No exaggerated anime mech ornament.
  - No floating impossible parts with no support logic.
  - No oversized “tech decal” clutter used as fake detail.
  - No clean white lab-future polish.

### Pure / Rune
- **Silhouette rules**
  - Read as ascendant, ritual, and pressure-held rather than soft mystic.
  - Keep figures vertical and centered, with cloak, ember, ash, or rune geometry framing the body.
  - Forms should feel refined by memory and rite, not random wizard layering.
  - Use controlled negative space: halos, rune gaps, floating ash bands, and poised hand language.
- **Materials**
  - Charred cloth, ritual metal, ash stone, ember glass, carved rune surfaces, soot-dark leather, ceremonial bindings.
  - Materials should feel sanctified, tempered, and carried through repeated rites.
  - Nothing should read plush, whimsical, or decorative for decoration’s sake.
- **Accent colors**
  - Core accent: violet rune light.
  - Support accents: fuchsia-violet bloom, ember orange, ash white.
  - Energy should feel internal and consequential, never candy-bright.
- **Iconography**
  - Runes, memory seals, ember circles, ascension marks, soul channels, ritual chambers.
  - Geometry should feel ancient but sharp, with clear hierarchy and intent.
- **FX behavior**
  - Ember lift, ash fall, rune ignition, contained flame, pressure halos, memory flare, slow resonance waves.
  - FX should gather, intensify, and resolve with ceremony.
- **Forbidden tropes**
  - No generic fantasy wizard robes.
  - No angelic purity cliché.
  - No soft blue “mana” shorthand.
  - No random spell particles with no rune source.
  - No playful occult styling.

## 2. Scene Kit

### Home / Command Citadel
- **Mood**
  - Centralized command, eclipse-lit authority, stillness before deployment.
- **Architecture cues**
  - Monumental city massing, strong horizon lines, elevated command platforms, central crest focus, disciplined framing.
- **Prop language**
  - Crest structures, interface pylons, command rails, mission terminals, restraint architecture, distant war infrastructure.
- **Lighting**
  - Dark base with red-violet atmospheric pressure and isolated faction glows entering from the sides.
  - The center must feel like the convergence point.
- **Environmental storytelling**
  - This is not a homey base. It is a war citadel that organizes identity, rank, and deployment.
  - Every set dressing choice should reinforce command, surveillance, and readiness.

### Bio Ruin Zone
- **Mood**
  - Predatory, contaminated, alive in the wrong way.
- **Architecture cues**
  - Broken district walls, collapsed biotech facilities, overgrown containment structures, growth pits, ruptured lab shells.
- **Prop language**
  - Specimen tanks, restraint hooks, field canisters, torn hazard panels, bone-like growth intrusions, sample crates.
- **Lighting**
  - Sick emerald pockets inside otherwise dark ruin space.
  - Light should look infected, not safely illuminated.
- **Environmental storytelling**
  - Show that research became hunting ground.
  - Spaces should imply extraction, failed control, and useful biomass still worth risking a squad for.

### Mecha Sanctum-Foundry
- **Mood**
  - Precision, heat discipline, industrial faith.
- **Architecture cues**
  - Bay doors, gantries, modular frame racks, weapon cradles, maintenance bridges, heavy vertical supports.
- **Prop language**
  - Chassis parts, tool arms, calibration rigs, coolant lines, stamped module cases, forge residue, diagnostic stations.
- **Lighting**
  - Cold cyan logic light over a darker steel body, with controlled amber heat only at work points.
- **Environmental storytelling**
  - This is where frames are tuned, repaired, and proven worthy.
  - The space should feel procedural and respected, like a machine chapel with manufacturing discipline.

### Pure / Rune Enclave
- **Mood**
  - Quiet pressure, ritual focus, memory awakening.
- **Architecture cues**
  - Circular chambers, stepped platforms, rune-carved thresholds, ember altars, ash-lined corridors, resonance monoliths.
- **Prop language**
  - Rite braziers, memory tablets, suspended seals, rune pillars, ceremonial chains, ash bowls, ember vessels.
- **Lighting**
  - Low dark surrounds with violet rune ignition and ember highlights.
  - Light should feel summoned and directional, not ambient wash.
- **Environmental storytelling**
  - These spaces are for attunement and ascent, not casual spirituality.
  - Every chamber should imply initiation, discipline, and the cost of resonance.

### Arena
- **Mood**
  - Formalized violence, prestige pressure, public trial.
- **Architecture cues**
  - Brutal combat framing, ring sightlines, audience walls, challenge gates, rank banners, impact-scarred surfaces.
- **Prop language**
  - Match pylons, rule displays, restraint barriers, trophy markers, maintenance scorches, combat reset devices.
- **Lighting**
  - Focused hot combat pools against deep shadow, with strong red-orange authority accents.
- **Environmental storytelling**
  - The Arena is not chaos; it is sanctioned proof.
  - Show repeat combat history, prestige systems, and institutional control over violence.

### Gate / Expedition Threshold
- **Mood**
  - Anticipation, transit risk, frontier initiation.
- **Architecture cues**
  - Threshold frames, sealed portals, charge arrays, route beacons, map pylons, departure platforms.
- **Prop language**
  - Travel seals, route markers, expedition crates, power couplings, clearance sigils, navigation consoles.
- **Lighting**
  - Dormant spaces stay dim; active spaces shift into energized green-cyan travel light.
  - Activation should feel expensive and deliberate.
- **Environmental storytelling**
  - This is where safe city logic ends and world-route danger begins.
  - Make route unlocks, travel cost, and mission consequence legible in the environment.

## 3. UI Law

### What the HUD should always do
- Keep the player anchored to three things at all times: **identity, condition, and resources**.
- Present the HUD as a command interface over a live war space, not as a floating game menu.
- Use framed panels, soft glass-black surfaces, and thin high-contrast edge lines.
- Maintain a strong center-versus-side composition: world focus in the middle, systems on the flanks, resources on the lower edge.
- Show path choice as a live state, not as a hidden submenu.

### What should never appear
- No bright flat-color blocks.
- No playful rounded-mobile UI language.
- No loot-box burst animations.
- No generic fantasy parchment, medieval ornament, or clean esports minimalism.
- No unreadable micro-text disguised as “tech.”
- No color usage that overrides the dark citadel base.

### Resource visibility rules
- Credits, core materials, and path-signature resources must stay visible in the persistent resource band.
- Resource chips should read in one glance: icon first, label second, count third.
- Do not bury active-path resources behind inventory screens.
- Rare or future resources may be hidden until relevant, but once a loop uses them, they join the persistent readability layer.
- Resource presentation must feel operational, not celebratory.

### Panel hierarchy rules
- Primary panels: identity, mission readiness, path selection, condition.
- Secondary panels: progression details, district states, route access, system summaries.
- Tertiary panels: flavor text, supporting stats, future-system placeholders.
- One panel should own each decision. Do not split a single choice across multiple equal-weight cards.
- The more actionable the panel, the higher its contrast and closer its placement to the player’s natural eye path.

### Path-accent usage rules
- Bio accent = emerald; Mecha accent = cyan; Pure/Rune accent = violet with controlled ember support.
- Path color is a guidance layer, not the full background.
- Use path accents on borders, progress fills, selected states, FX glows, and key chips.
- Keep base surfaces dark and neutral so path color feels like activation, not wallpaper.
- Never mix multiple faction accents at equal strength in one focal panel unless the point is conflict or convergence.

## 4. Progression Presentation Law

### How ascension should look
- Ascension must feel ceremonial and identity-deepening.
- Build it as a controlled sequence: gather pressure, ignite the faction language, resolve into a more refined silhouette/state.
- Prefer vertical motion, crest/rune/core alignment, and environmental participation over showering the screen with rewards.
- The player should feel transformed, not paid.

### How rank-up should look
- Rank-up should feel officially recognized by the system.
- Use stamped confirmation, panel reinforcement, title elevation, and a brief surge through the active faction accent.
- Keep timing short and authoritative.
- Favor “status updated” over “jackpot won.”

### How war/meta should look
- Meta progression should present as campaign pressure, district control, route expansion, and faction commitment.
- Use maps, thresholds, command overlays, unlock states, and institutional markers.
- Every meta change should suggest the world shifting, not just the account number rising.

### How to avoid “reward payout” feeling
- Do not lead with coin bursts, chest openings, confetti, or slot-machine pacing.
- Show outcome through consequence: new route online, system state changed, rank bar advanced, faction panel intensified, specimen archived, module installed.
- Use restrained audio/visual punctuation if implemented later; the dominant message is progression, not prize dispensing.
- Rewards should feel earned through operation logs, combat reports, rites completed, and field recovery.

## 5. Build Priorities

1. **Lock faction silhouette sheets** for Bio, Mecha, and Pure/Rune so all future character art follows one readable shape language.
2. **Define faction material boards** with approved surface families, damage logic, and trim usage.
3. **Create the command citadel key art pack** to anchor the whole product’s mood and UI backdrop.
4. **Build the scene kit paintovers** for Bio Ruin Zone, Mecha Sanctum-Foundry, Pure/Rune Enclave, Arena, and Gate threshold.
5. **Standardize faction accent usage across UI** so borders, glows, progress bars, and selected states all obey the same law.
6. **Replace placeholder resource/icon language** with faction-aware operational iconography and clearer resource grouping.
7. **Design progression presentation boards** for ascension, rank-up, route unlock, and meta-state change using non-payout motion logic.
8. **Produce environmental storytelling callout sheets** for each district so prop teams know what history each space must imply.
9. **Build UI hierarchy templates** for primary, secondary, and tertiary panel types before adding more features.
10. **Assemble implementation-ready handoff pages** that pair each law sheet with examples, do/don’t references, and naming aligned to current game terms.

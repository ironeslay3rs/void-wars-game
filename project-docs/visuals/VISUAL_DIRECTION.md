\# VISUAL DIRECTION



\## Core Rule

Field = gameplay clarity

UI = identity and detail



\## Field Rules

\- No UI panels dominating the screen

\- Entities must feel grounded

\- Clear silhouettes

\- Readable spacing

\- No clutter



\## UI Rules

\- Dark theme

\- High contrast icons

\- Clean spacing

\- Minimal text

\- **Mockup reference pack:** high-fidelity layout targets live in `UI_MOCKUP_REFERENCE.md` (copy PNGs into `visuals/mockups/`). Use for composition and tone; ship art only via `ASSET_PIPELINE.md`.



\## Visual Layers

1\. Background (map)

2\. Entities (player + mobs)

3\. Effects (hits, drops)

4\. UI overlays (HUD)



\## DO NOT

\- mix portrait art into field

\- use JPG assets with no transparency

\- overload screen with icons

\# CORE LOOP



\## Main Loop

Deploy → Hunt → Result → Recover → Repeat



\## Field Loop

Move → Target → Attack → Kill → Collect



\## Design Rules

\- Rewards must feel earned

\- Survival pressure must exist

\- Player must always know what to do next



\## AFK Rule

\- AFK = source of truth

\- Realtime = bonus layer



\## Clarity Rule

Every screen must answer:

\- What happened?

\- What changed?

\- What do I do next?
